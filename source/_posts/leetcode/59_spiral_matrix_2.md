## 题目([leetcode-59](https://leetcode.com/problems/spiral-matrix-ii/))
给定正整数n, 以螺旋方式生成一个`n x n`矩阵, 矩阵元素为1至$n^2$.  
例:
```
Input: n = 3
Output: [[1,2,3],[8,9,4],[7,6,5]]

Input: n = 1
Output: [[1]]
```
限制: $1 <= n <= 20$
